# Roastly Cafe

A premium, handcrafted web experience for an artisanal coffee shop.

## Overview

Roastly Cafe is a modern, performant, and visually stunning interactive website built to showcase premium coffee and artisanal pastries. The design focuses on a handcrafted aesthetic, featuring custom typography, hand-drawn botanical illustrations, smooth scroll animations, and a rich, warm color palette.

## Tech Stack

- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion (`motion/react`)
- **Icons:** Lucide React
- **Typography:** Playfair Display (Headings), Caveat (Accents), Inter (Body)

## Key Features

- **Premium Aesthetic:** Warm, parchment-style background with a subtle grain texture.
- **Custom Assets:** JS-driven coffee bean cursor, hand-drawn vector accents (leaves, coffee beans).
- **Smooth Animations:** Scroll-triggered reveals, subtle hover states, and atmospheric CSS animations.
- **Responsive Design:** Fully mobile-optimized layout with a clean hamburger menu.
- **Dynamic Routing:** Smooth scrolling navigation mapping directly to section IDs.

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/Karunya-Muddana/Roastly_Website.git
   ```
2. Navigate to the project directory
   ```bash
   cd roastly-cafe
   ```
3. Install dependencies
   ```bash
   npm install
   ```

### Running Locally
To start the Vite development server:
```bash
npm run dev
```
Navigate to `http://localhost:3000` in your browser.

## Deployment

This project is fully optimized for immediate deployment on platforms like Vercel or Netlify.

### Build for Production
```bash
npm run build
```
The output will be generated in the `dist` folder.
